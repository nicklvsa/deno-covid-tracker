# Deno Covid Tracker

### Requirements:
- make sure latest version of deno is installed
- run `deno run -A --unstable index.ts ---country {ENTER_COUNTRY_NAME}`

## Info:
- API used: https://rapidapi.com/Gramzivi/api/covid-19-data
